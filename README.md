# Gizzi Code

> **Workspace-aware AI assistant CLI**
> AI-powered terminal interface for the Allternit ecosystem. Natural language commands, multi-provider support, and local-first execution.

## Installation

```bash
# macOS / Linux
curl -fsSL https://install.gizziio.com/install | bash

# Windows (PowerShell)
irm https://install.gizziio.com/install.ps1 | iex

# npm
npm install -g @allternit/gizzi-code
```

## Quick Start

```bash
# Start interactive TUI
gizzi-code

# Run a single command
gizzi-code run "Explain this codebase"

# View help
gizzi-code --help
```

## Features

- **Multi-Provider** — Claude, OpenAI, Kimi, Gemini, and more
- **Computer Use** — Browser and desktop automation
- **Multi-file Editing** — Refactor across your entire codebase
- **Git Integration** — Generate commits, review diffs, manage PRs
- **Skill System** — Extensible capabilities via the Allternit SDK
- **Local-First** — Runs locally with secure sandboxed execution
